# AI---Chatbot

A simple chatbot built using Streamlit and LangChain with the Groq API.

## Features

- Chat interface
- Conversation memory
- Powered by Llama 3.3 70B
- Built with Streamlit

## Installation

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```env
GROQ_API_KEY=your_api_key
```

Run the app:

```bash
streamlit run bot.py
```
